# 🎓 AI Course Creator

A web-based application that automatically generates complete, structured courses from any topic using Generative AI — with instant PDF export.

🔗 **Live Demo:** [devikrishna94191-tech.github.io/ai-course-creator](https://devikrishna94191-tech.github.io/ai-course-creator/)

---

## ✨ Features

- **Instant Course Generation** — Enter any topic and get a fully structured course with modules, lessons, and content
- **PDF Export** — Download the generated course as a clean, formatted PDF with one click
- **No Installation Required** — Runs entirely in the browser, no backend or server setup needed
- **Powered by Claude AI** — Uses Anthropic's Claude API for intelligent, context-aware course creation

---

## 🚀 How to Use

1. Open the [live app](https://devikrishna94191-tech.github.io/ai-course-creator/)
2. Enter a topic you want to learn about (e.g. "Python for Beginners", "Digital Marketing Basics")
3. Click **Generate Course**
4. Browse the generated modules and lessons
5. Click **Export as PDF** to download your course
